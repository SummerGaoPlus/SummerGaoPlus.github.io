# 1.碳素结构钢的分级

碳素结构钢又称为普碳钢，钢号按国家标准《碳素结构钢》GB/T 700-2006，将碳素 结构钢屈服强度的下限值分为四个级别，对应为Q195、Q215、Q235和Q275，其中Q 代表屈服强度，数字为屈服强度的下限值，数字后面标注的字母A、B、C、D表示钢材的质量等 级，即硫、磷质量分数不同，A级钢中的硫、磷含量最高，D级钢中硫、磷含量最低。
