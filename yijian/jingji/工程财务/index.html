# 1Z102000 工程财务

工程财务依据财务会计基础，结合施工企业和工程项目特点实施财务会计和财务管理。 工程财务主要运用货币价值形态，通过核算、监督、分析和预测等手段，综合反映施工企 业资产、负债、现金流量等财务状况，以及工程项目实施工程中的劳动投入、资产占用、 财务收支和利润情况，为施工企业和工程项目提供管理和决策依据。从事工程项目管理岗 位工作的建造师掌握必要的工程财务原理和方法，明确工程财务制度、规定和要求，有利 于其在工作中严格财经纪律、有效利用资产、控制财务收支、提高经济效益。

# 1Z102010 财务会计基础

财务会计是按会计准则和会计的制度要求，对过去已发生的经济活动通过记账、算账 和报账等专门方法向单位外部关系人提供单位的财务状况、经营成果和资金变动情况等有 关信息的会计。

# 1Z102011

企业财务会计主要是对企业已经发生的交易或事项，通过确认、计量和报告程序进行 加工处理，并借助以财务报表为主要内容的财务报告，向外部利害关系人提供以财务信息 为主的经济信息。企业财务会计具有以下特点：

  1. 提供的信息侧重于过去信息。现代会计虽然也进行一些财务预测，如成本预测、 现金流量预测等，但其主要是对企业已经发生的交易或事项进行核算和报告、提供过去的 经营成果、财务状况和现金流量信息。
  2. 主要为外部利害关系人服务。企业经营管理者可以依据财务会计信息进行经营管 理决策，但财务会计信息本身主要为企业外部利害关系人服务，外部利害关系人主要有投 资人、债权人、政府及其有关部门、社会公众。财务会计对外提供的信息反映了企业与投 资者、债权人的利益关系，是投资者和债权人做出合理决策的重要依据；是投资人和社会 公众监督和衡量企业经营者经营企业资源、履行社会责任绩效的依据；还是政府部门进行 宏观经济管理的重要信息来源。
