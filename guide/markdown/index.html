  2. 在开头的时候，先输入如下片段，然后紧跟着输入文本即可。分号也不要掉。

# 十二、在Markdown中打出上标、下标和一些特殊符号

# 上标

上标的HTML标签的是< sup >，所以如果要打上标的话就用以下格式：

<sup>xxx</sup>
1

其中xxx表示上标的内容，看个例子：
我现在想写一个公式：n的平方等于n+1，写法如下：

n<sup>2</sup>=n+1
1

显示效果就是：
n2=n+1

# 下标

下标的标签是< sub >，同理我们来实现一个例子： a=log2b 写法如下：

a=log<sub>2</sub>b
1

a=log2b
还有一些特殊的、键盘上没有的符号也可以打出来，比如注册商标的符号：立白®
